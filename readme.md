# Internship Task – Project README

## Project Overview
This project was created as part of an internship selection task.  
It demonstrates basic web development skills including project structuring, frontend implementation, server-side routing, and handling static files.  
The goal is to showcase:
- Understanding of web application flow  
- Ability to work with Node.js and Express  
- Clean and organized code structure  
- Proper use of tools and development practices

## Tools & Technologies Used
- HTML, CSS, JavaScript – for building the frontend  
- Node.js – backend runtime environment  
- Express.js – to handle server logic and routing  
- Git & GitHub – for version control and submission  
- npm – Node package manager

## How to Run or View the Project

Follow these steps to set up and run the project locally:

### 1. Clone the Repository
```bash
git clone https://github.com/HimanshuKabra01/SuPrazo
cd SuPrazo
```
### 2. Install Dependencies
```bash
npm install
```

### 3. Edit the server.js file
#### Add the following code
```bash
  app.listen(PORT, () => {
    console.log(`Server is running on ${PORT}`)
  })
```
